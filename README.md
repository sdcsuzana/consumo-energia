# Cálculo de gasto de um produto em kwh ⚡📺

Projeto desenvolvido para que o cliente possa **calcular quantos kwh ele gasta por mês com um eletrodoméstico específico de sua preferência**.

Programa feito em liguagem python.

<img align="center" alt="Python" height="40" width="40"
src="https://cdn.jsdelivr.net/gh/devicons/devicon/icons/python/python-original.svg">

A fórmula usada pegar a potência, fazer vezes o número de dis médio do mês (30) e dividir por 1000, este último usado para conversão de watts para kilowatts.

O programa solicita o nome do aparelho, a potência, o tempo médio de uso em horas e dá como resultado o **consumo mensal**.
